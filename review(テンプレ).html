  1. マシンの電源が入っている場合、2 つ目の条件も実行して、

おはようございます。提供されたコードにはいくつかの構文エラーがあります。以下の修正が必要です：


  switch (machineActive) {
    case true:
      machineResult = 'でんげんON';
      break;
    default:
      machineResult = 'でんげんをいれてください';
  }

修正点は次の通りです：

  1. case trueの後にはコロン（:）が必要です。
  2. case節の後にはステートメント（ここではmachineResult = 'でんげんON';）が必要です。
  3. case節の最後には必ずbreak文を追加する必要があります。

修正を行うことで、コードが正しく動作するはずです。お気をつけてお試しください。
