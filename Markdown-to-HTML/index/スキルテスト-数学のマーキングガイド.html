JavaScript math marking guide

タスクの目的は、Learn Web Development on MDN の Basic math in JavaScript - numbers and operators レッスンでカバーされている JavaScript 機能を理解していることを示すことです。

注：コードにエラーがある場合、答えを見つけるために、ページの結果パネルに出力されます（ダウンロード版の場合は、ブラウザのJavaScriptコンソールに出力されます）。

Task 1

このタスクでは、数値を含む4つの変数を作成する必要があります。最初の2つは足して格納する必要があり、4つ目は3つ目から引いて結果を格納する必要があります。これらの演算の2つの結果を掛け合わせて、別の結果である48を作成する必要があり、これは変数finalResultに格納する必要があります。

最後に、この数字が偶数か奇数かをチェックする計算を書き、その結果を evenOddCheck に格納する必要があります。ここで必要な演算子はモジュロ(%)です。モジュロ2の値が0になれば、偶数であることがわかります。

答えはこのようになるはずです：

const number1 = 4;
const number2 = 8;
const number3 = 12;
const number4 = 8;

const additionResult = number1 + number2;
const subtractionResult = number3 - number4;

finalResult = additionResult * subtractionResult;

evenOddResult = finalResult % 2;

このタスクは、変数の命名能力もテストします。変数がベストプラクティス（キャメルケースの下、説明的、長すぎない）に従った賢明な名前である場合、追加のマークが付けられます。

Task 2

この記事の2つ目の課題では、演算子の優先順位を考慮し、最初の2つの計算を期待通りの結果にするために括弧を使用する必要があります。

次に、理想的には、乗算代入演算子を使って2つの結果を掛け合わせ、その積を result に代入して戻す必要があります。

その後、数値で利用できる toFixed() メソッドを使って result を小数点以下2桁に丸め、その積を finalResult 変数に格納する必要があります。

最後に、typeofを使って、変数に文字列が含まれているかどうかをテストすることができます。Number()を使って、finalResultを数値型に変換する必要があります。この最終的な数値は finalNumber` という変数に格納する必要がある。

結果はこのような感じになるはずです：

let result = (7 + 13) / (9 + 7);
let result2 = 100 / (2 * 6);

result *= result2;

const finalResult = result.toFixed(2);

const finalNumber = Number(finalResult);

Task 3

この最後の課題では、比較演算子を使う能力が試されます。ここでは、数学的に問題文の記述を証明したり反証したりすることを示すテストをいくつか書く必要があります。

テスト結果は、それぞれ weightComparison、heightComparison、pwdMatch という変数に格納する必要があります。

コードは次のようになります：

const weightComparison = eleWeight < mouseWeight;

const heightComparison = ostrichHeight > duckHeight;

const pwdMatch = pwd1 === pwd2;
