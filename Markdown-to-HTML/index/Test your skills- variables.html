変数1

このタスクでは、以下のことを行ってください：

  • myNameという名前の変数を宣言する。
  • myNameを適当な値で初期化する（別の行で）（実際の名前を使うか、他のものを使うか）。
  • myAgeという変数を宣言し、同じ行にある値で初期化する。

以下のライブコードを更新して、完成した例を再現してみてください：


// Add your code here

// Don't edit the code below here!

section.innerHTML = ' ';
const para1 = document.createElement('p');
para1.textContent = myName;
const para2 = document.createElement('p');
para2.textContent = myAge;
section.appendChild(para1);
section.appendChild(para2);

変数 2

このタスクでは、既存のmyName変数に格納されている値を自分の名前に修正するために、新しい行を追加する必要があります。

以下のライブコードを更新して、完成した例を再現してみてください：


// Add your code here

// Don't edit the code below here!

section.innerHTML = ' ';
const para = document.createElement('p');
para.textContent = myName;
section.appendChild(para);

変数 3

最後の課題です。この場合、既存のコードが提供されますが、このコードには2つのエラーがあります。結果パネルには、Chrisという名前と、Chrisが20年後に何歳になっているかという文が出力されているはずです。この問題を解決し、出力を修正するにはどうすればよいでしょうか？

以下のライブコードを更新して、完成した例を再現してみてください：


// Add your code here

const myName = 'Default';
myName = 'Chris';

let myAge = '42';

// Don't edit the code below here!

section.innerHTML = ' ';
const para1 = document.createElement('p');
const para2 = document.createElement('p');
para1.textContent = myName;
para2.textContent = `In 20 years, I will be ${myAge + 20}`;
section.appendChild(para1);
section.appendChild(para2);
