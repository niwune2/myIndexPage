模範解答-3

const names = ['Chris', 'Li Kang', 'Anne', 'Francesca', 'Mustafa', 'Tina', 'Bert', 'Jada']
const para = document.querySelector('p');

function random(min, max) {     //1
  const num = Math.floor(Math.random() * (max - min)) + min;    //2
  return num;   //3
}

function chooseItem(array) {    //4
  const choice = array[random(0, array.length)];    //5
  return choice;    //6
}
para.textContent = chooseItem(names);   //7
  1. //1. ランダムな数値を生成する関数
    1. 引数にmin,maxを使う
  2. //2.変数numに、function 2で使ったランダムな整数のコード[Math.floor(Math.random()*names.length)]を書き換え、max とminの間の数値でランダムな数値を生成できるようにする
  3. //3.変数numは"設定された範囲内の中の整数をランダムに返す式"であり、random()が呼び出されるときはこの変数の値が用いられる
  4. //4.配列から一つの名前を取り出す関数chooseItem(array)
    1. 引数arrayは関数内で、その値を処理するために用いる。
    2. //7.でこの関数が呼び出されているが、括弧に変数namesを入れることで"names内の中から一つを選ぶ”ことができる
    3. namesのような配列の変数が複数ある場合、そのまま括弧の中を置き換えるだけでよく、わざわざ一つの変数に対応する関数を記述する必要がなくなる
  5. //5.[random(0, array.length)]は上記の関数の呼び出し
  6. //6.変数choiceの値をchooseItem関数の値として呼び出すためのreturn

GPTに確認-function4

一つ確認をお願いします。

function isShort(name) {    //1
  return name.length < 5;   //2
}
const shortNames = names.filter(isShort);   //3

上のコードと

const shortNames = names.filter(isShort=(name)=>name.length<5);

このコードは同じ機能でしょうか？

いいえ、それらのコードは異なる機能を持ちます。

最初のコードでは、関数isShortが定義されており、引数nameの長さが5未満かどうかをチェックしています。そして、namesという配列に対してfilterメソッドを使い、関数isShortをフィルター関数として渡しています。この結果、shortNames配列にはnames配列から長さが5未満の要素だけがフィルタリングされます。

一方、2番目のコードでは、isShortという名前のアロー関数が定義されています。このアロー関数は単一の引数nameを取り、その長さが5未満であるかどうかを評価しています。そして、このアロー関数自体をisShortという変数に代入しています。その後、names配列に対してfilterメソッドを使用し、アロー関数isShortをフィルター関数として渡しています。これにより、同じく長さが5未満の要素だけがフィルタリングされ、shortNames配列に格納されます。

結果として、2つのコードは同じ機能を実現していますが、関数の定義方法や関数の代入方法が異なる点に注意してください。

どちらも"namesから5文字未満の名前を抜き取る"という点では共通しているのでしょうか

はい、正確に言えば、両方のコードは同じ目的を持っています。両方のコードは、namesという配列から5文字未満の名前をフィルタリングして取得することを目的としています。

最初のコードでは、関数isShortを定義して、名前の長さが5未満かどうかを判定しています。そして、その関数をfilterメソッドの引数として渡しています。filterメソッドは、与えられたフィルター関数を使用して配列をフィルタリングし、条件を満たす要素だけを取得します。

2番目のコードでは、アロー関数を使って直接フィルター関数を定義しています。このアロー関数も同様に、名前の長さが5未満かどうかを判定します。そして、そのアロー関数をfilterメソッドの引数として渡しています。

したがって、両方のコードは同じ結果を得るために同じフィルタリングの機能を実行しています。ただし、関数の定義と関数の代入の方法が異なるため、コードの構造や文法が異なることに注意してください。
