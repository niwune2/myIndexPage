アクティブ学習: もっとたくさんの色から選ぶ！

const select = document.querySelector('select');
const html = document.querySelector('.output');

select.onchange = function() {//0
  const choice = select.value;//0

  // ここに SWITCH ステートメントを書く
  switch(choice){ //1
    case 'white'://2
        update('white','blue');//3
        break;//4
    case 'black'://5
        update('black','gray');//6
        break;//7
    case 'purple'://8
        update('purple','red');//9
        break;//10
    case 'yellow'://11
        update('yellow','orange');//12
        break;//13
    case 'psychedelic'://14
        update('lime','purple');//15
        break//16
  }
}
function update(bgColor, textColor) {
  html.style.backgroundColor = bgColor;
  html.style.color = textColor;
}

コードレビュー - 色選び

  1. choice変数は、select要素で取得した値を格納している
  2. switchステートメントで、choice変数で取得した値をもとに、それぞれ処理を行う
  3. whiteを選択した場合
  4. update関数を呼び出し、ページに色の変更を反映する。
    1. update関数は、背景色と文字色を変更するための関数
  5. 条件に当てはまった場合はbreakで止まり、処理が実行される。当てはまらない場合は、そのまま下の条件を探す
  6. ~ 16.まで同じ
  7. いずれの値とも一致しなかった場合はdefault句の後に記述された処理を実行する。この句が不要である場合は省略可能。

コードレビュー - 条件文4

  1. //1: machineActiveがtrueかfalseかに応じて処理を決める条件文

  3. //3: machineActiveがtrueであったときの処理を記述する
  4. //4: machineResultにメッセージを格納し、表示させる
  5. //5: 三項演算子を使った条件文
    1. 構文: ( 条件式 ) ? こちらのコードを実行する : 代わりにこちらのコードを実行する
    2. 例1:let greeting = ( isBirthday ) ? '誕生日おめでとうございます！' : 'おはようございます。';
    3. 例2:( select.value === 'black' ) ? update('black','white') : update('white','black');
    4. pwdResultに格納させるため、=を書く
    5. 括弧()内に条件式を書く
    6. ?を挟む
    7. trueの場合は右に、falseの場合は左に記述する。間にはコロン:を挟む
