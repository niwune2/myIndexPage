Events 2

次に、キーボードイベントについて見ていきます。この評価に合格するには、キーボードのWASDキーが押されたときに、提供されたキャンバスの周りを円を移動させるイベントハンドラを構築する必要があります。円の描画は関数 drawCircle() で行い、入力として以下のパラメータを受け取ります：

  • x — 円のx座標を指定する。
  • y — 円のy座標を指定する。
  • size — 円の半径を表す。
const canvas = document.querySelector('canvas');
const ctx = canvas.getContext('2d');

function drawCircle(x, y, size) {
  ctx.fillStyle = 'white';
  ctx.fillRect(0, 0, canvas.width, canvas.height);

  ctx.beginPath();
  ctx.fillStyle = 'black';
  ctx.arc(x, y, size, 0, 2 * Math.PI);
  ctx.fill();
}

let x = 50;
let y = 50;
const size = 30;

drawCircle(x, y, size);
